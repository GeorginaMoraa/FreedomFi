FreedomFi ...

A simple, responsive financial awareness web application designed to promote financial freedom through structured budgeting principles like the 50/30/20 rule.

📌 Overview

FreedomFi is a static web application that helps users understand and apply smart budgeting strategies to achieve financial stability and independence.

The platform focuses on:
	•	Financial discipline
	•	Budget allocation
	•	Awareness of spending habits

⸻

🎯 Features
	•	📊 Budgeting Guidance
Learn how to allocate income using proven financial rules (e.g., 50/30/20).
	•	💡 Financial Awareness
Encourages smarter decision-making around spending and saving.
	•	📱 Responsive Design
Works seamlessly across mobile, tablet, and desktop devices.
	•	⚡ Lightweight & Fast
Built with pure HTML, CSS, and JavaScript — no heavy frameworks.

⸻

🛠️ Tech Stack
	•	HTML5 – Structure
	•	CSS3 – Styling and layout
	•	JavaScript – Interactivity
